# Bots de Automação — Nexora Academy

Esta pasta contém bots e scripts para automação de tarefas no projeto Nexora Academy.

## Bots Disponíveis

- **Jira Epic Generator** (`jira-epic-generator/`):
  - Criação automática de épicos e histórias de usuário no Jira usando IA (Claude API).
- **Criador de Tasks Jira** (`create_aicma_tasks.py`):
  - Script Python para criar tasks AICMA no Jira via API REST.
- **Bot de Testes Robot Framework** (`robot_test_agent.py`):
  - Executa testes automatizados usando Robot Framework.
- **Bots GitHub (em desenvolvimento):**
  - PR Review, Commit Assistant, PR Creator, Quality Gate.

## Como Usar

Consulte o README de cada subpasta ou script para instruções específicas de execução e configuração.

## Requisitos

- Node.js (para bots em TypeScript)
- Python 3 (para scripts Python)
- Acesso à API do Jira e/ou GitHub conforme o bot

## Estrutura

- Cada bot possui documentação própria e exemplos de uso.
- Scripts e agentes podem ser integrados ao CI/CD ou executados manualmente.
